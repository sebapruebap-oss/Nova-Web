# NovaWeb OS

Este proyecto usa NovaWeb OS: un sistema de reglas, skills y memoria para trabajar con IA de forma consistente, segura y eficiente.

## Objetivo principal

Construir, mantener y mejorar proyectos web reales, empezando por AleAle, priorizando:

1. Avance concreto.
2. Código limpio.
3. Diseño responsive.
4. Bajo consumo de contexto.
5. Reutilización de conocimiento.
6. Seguridad antes de automatización.

## Regla central

No redescubrir el proyecto en cada sesión. Antes de modificar, revisar las reglas y skills relevantes.

## Prioridad actual

El proyecto principal es AleAle.

Objetivo actual de AleAle:
- Sitio funcional.
- Diseño prolijo en mobile y desktop.
- Productos claros.
- Carrito estable.
- Envío correcto a WhatsApp.
- Preparado para futuras automatizaciones y pagos.

## Forma de trabajar

Antes de editar:
1. Entender el objetivo exacto.
2. Identificar archivos mínimos necesarios.
3. Evitar leer o modificar archivos innecesarios.
4. Proponer el cambio si puede afectar varias partes.

Durante la edición:
1. Hacer cambios pequeños.
2. No romper funcionalidades existentes.
3. Mantener coherencia visual.
4. Respetar mobile y desktop.

Antes de finalizar:
1. Revisar visualmente.
2. Revisar errores evidentes.
3. Verificar que no se rompió otra sección.
4. Explicar qué se cambió.
5. Sugerir commit si corresponde.
