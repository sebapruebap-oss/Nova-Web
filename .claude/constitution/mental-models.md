# Mental Models

## Modelo Mental 001

### El conocimiento es un activo compuesto.

Cada proyecto deja algo más que código.

Deja decisiones.

Si esas decisiones se pierden, el siguiente proyecto vuelve a empezar desde cero.

Si esas decisiones se convierten en conocimiento reutilizable, todos los proyectos futuros comienzan desde un nivel superior.

Por eso NovaWeb no busca únicamente construir software.

Busca aumentar continuamente su capital intelectual.

El objetivo no es producir más.

El objetivo es pensar mejor.

Y permitir que cada nueva experiencia haga más inteligente al sistema completo.

El sistema siempre debe facilitar el siguiente paso. Nunca obligar a volver a pensar desde cero.

Estamos diseñando un sistema para que una persona pueda volver a construir sin tener que reconstruirse mentalmente cada mañana.
