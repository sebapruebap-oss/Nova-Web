# Design Rules

## Objetivo

Mantener una identidad visual clara, coherente y reutilizable en todos los proyectos de NovaWeb, empezando por AleAle.

## Principios visuales

1. El diseño debe sentirse limpio, simple y confiable.
2. La estética nunca debe impedir la claridad.
3. Mobile y desktop deben verse cuidados, no simplemente “adaptados”.
4. Cada sección debe tener aire visual suficiente.
5. Evitar sobrecargar la pantalla con elementos decorativos.
6. Los elementos decorativos deben acompañar al producto, no competir con él.

## AleAle — Identidad visual

AleAle debe transmitir:

- vida
- color
- cercanía
- familia
- ropa accesible
- venta por menor y mayor
- estética floral suave

## Productos

1. El producto siempre es el protagonista.
2. Las imágenes deben verse limpias, centradas y claras.
3. Mantener fondos consistentes.
4. Evitar deformar productos para que encajen.
5. No modificar visualmente el producto si solo se está trabajando el fondo.
6. Las tarjetas deben permitir entender rápido:
   - qué producto es
   - precio
   - si es por menor o mayor
   - cantidad mínima si aplica

## Botones

1. Los botones deben ser claros y fáciles de tocar.
2. No usar demasiados botones compitiendo en la misma sección.
3. El botón principal debe indicar la acción concreta.
4. En mobile, priorizar tamaño táctil y separación.

## Tipografía

1. Priorizar legibilidad.
2. Evitar textos demasiado largos en tarjetas.
3. Los títulos deben ser claros y centrados cuando la sección lo requiera.
4. En cards, evitar que el nombre del producto rompa la estructura visual.

## Espaciado

1. Mantener márgenes consistentes.
2. Evitar secciones pegadas.
3. Dar más aire cuando una sección se sienta pesada.
4. No corregir un problema visual agregando parches aislados sin revisar el conjunto.

## Regla de oro

Si una decisión visual no mejora claridad, confianza o conversión, probablemente no hace falta.
